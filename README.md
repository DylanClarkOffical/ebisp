# Ebisp

Embedded Lisp

## Quick Start

```
$ gcc -o build build.c   # bootstrap the build
$ ./build                # build the entire project
$ ./repl                 # run the Ebisp repl
$ ./ebisp_test           # run the Unit Tests
```
